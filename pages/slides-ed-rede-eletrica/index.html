A Estrutura

Foi desenvolvido um Grafo Direcionado e Ponderado genérico, adaptado para modelar uma rede de distribuição de energia.

Vertice<TIPO>

Representa os pontos da rede: Usinas, Subestações e Postes. Armazena listas de arestas de entrada e saída.

Aresta<TIPO>

Representa os cabos conectando os pontos. Possui um peso (Double) que define a distância em km.

Grafo<TIPO>

Gerencia a coleção de vértices e arestas. Controla conexões, mapeamento toString() e o algoritmo de BuscaEmLargura().

BuscaEmLargura() 
public void BuscaEmLargura() {
    ArrayList<Vertice<TIPO>> marcados = new ArrayList<Vertice<TIPO>>();
    ArrayList<Vertice<TIPO>> fila = new ArrayList<Vertice<TIPO>>();

    Vertice<TIPO> atual = this.vertices.get(0);

    marcados.add(atual);

    System.out.println("[INICIO] Ponto de partida: [ " + atual.getDado() + " ]");

    fila.add(atual);

    while (fila.size() > 0) {
        Vertice<TIPO> visitado = fila.get(0);

        for (int i = 0; i < visitado.getArestasSaida().size(); i++) {
            Vertice<TIPO> proximo = visitado.getArestasSaida().get(i).getFim();

            if (!marcados.contains(proximo)) {
                marcados.add(proximo);

                System.out.println("   ---> Energia fluiu para: [ " + proximo.getDado() + " ]");

                fila.add(proximo);
            }
        }
        fila.remove(0);
    }
}

Busca em Largura (BFS)

O algoritmo BFS (Breadth-First Search) foi implementado com uma Fila (ArrayList) e uma lista de marcados para evitar loops.

Ele simula perfeitamente a lógica de propagação física: a energia elétrica flui em ondas, alimentando primeiro as subestações diretas, antes de alcançar os postes mais distantes.
